How LIFO Cache Replacement Works

  • Sequential Storage: First value → B0, second → B1, third → B2, fourth → B3
  • LIFO (Last-In First-Out): Replaces the most recently added item when cache is full
  • Cache Hit: Value found in cache → Green highlight on THAT block only
  • Cache Miss: Value not found:
    • If cache has empty blocks → Added to next empty block (Purple highlight)
    • If cache is full → Replaces Last-In block (Red highlight on replaced block)
  • LIFO Stack: Tracks insertion order with most recent at top
